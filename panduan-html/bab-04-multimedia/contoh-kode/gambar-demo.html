Halaman ini mendemonstrasikan berbagai penggunaan tag <img> dalam HTML dengan contoh praktis.

1. Gambar Dasar dengan Atribut Wajib

Setiap gambar harus memiliki atribut src dan alt:

<img src="URL_GAMBAR" alt="Deskripsi gambar">

Gambar Dasar
Dengan src dan alt yang benar

2. Mengatur Ukuran Gambar

Gunakan atribut width dan height, atau CSS untuk kontrol yang lebih baik:

Ukuran Tetap
width="300" height="200"

<img width="300" height="200">

Responsif
max-width: 100%; height: auto;

<img style="max-width: 100%; height: auto;">

4. Gambar sebagai Link

Gambar dapat dijadikan link dengan membungkusnya dalam tag <a>:

Gambar Link
Klik gambar untuk membuka Google

<a href="https://www.google.com" target="_blank"> <img src="gambar.jpg" alt="Deskripsi"> </a>

5. Lazy Loading untuk Performa

Gunakan loading="lazy" untuk memuat gambar hanya saat dibutuhkan:

Scroll ke bawah untuk melihat gambar lazy loading...

Lazy Image 1

Lazy Image 2

Lazy Image 3

<img src="gambar.jpg" alt="Deskripsi" loading="lazy">

6. Gambar Responsif dengan srcset

Gunakan srcset dan sizes untuk gambar yang optimal di berbagai ukuran layar:

Responsive Image
Ukuran menyesuaikan layar perangkat

<img src="default.jpg" srcset="small.jpg 300w, medium.jpg 600w, large.jpg 900w" sizes="(max-width: 300px) 100vw, (max-width: 600px) 50vw, 33vw" alt="Responsive image">

7. Penanganan Error Gambar

Ketika gambar tidak dapat dimuat, alt text akan ditampilkan:

Gambar Tidak Ditemukan
Alt text ditampilkan sebagai fallback

HTTP Error
Server mengembalikan error 404

8. Best Practices

‚úÖ Yang Harus Dilakukan:

  • Selalu gunakan atribut alt yang deskriptif
  • Optimasi ukuran file gambar sebelum upload
  • Gunakan format yang tepat (JPEG untuk foto, PNG untuk grafis)
  • Implementasikan lazy loading untuk gambar di bawah fold
  • Gunakan responsive images dengan srcset
  • Berikan width dan height untuk menghindari layout shift

‚ùå Yang Harus Dihindari:

  • Menggunakan gambar tanpa alt text
  • Upload gambar dengan ukuran file yang terlalu besar
  • Menggunakan format yang tidak sesuai
  • Hardcode ukuran yang tidak responsive
  • Lupa compress gambar sebelum upload
  • Menggunakan gambar copyrighted tanpa izin
