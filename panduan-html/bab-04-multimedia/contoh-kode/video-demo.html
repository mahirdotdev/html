Tag <video> memiliki berbagai atribut untuk mengontrol tampilan dan perilaku:

Atribut Deskripsi Nilai Contoh Penggunaan

muted Memulai dalam kondisi mute boolean muted
preload Strategi loading video none, metadata, auto preload="metadata"

üí° Best Practice Format:

Untuk kompatibilitas maksimum, sediakan format dalam urutan ini:

  2. WebM (VP9) - untuk browser modern dengan efisiensi tinggi
  3. OGV - fallback untuk browser lama

9. Optimasi Performance Video

üì¶ Ukuran & Kompresi

  • Resolusi: 720p untuk web standard
  • Bitrate: 1-3 Mbps untuk 720p
  • Codec: H.264 untuk kompatibilitas
  • Compression: Handbrake, FFmpeg

‚ö° Loading Strategy

  • preload="none" - Hemat bandwidth
  • preload="metadata" - Balance
  • preload="auto" - UX terbaik
  • Lazy loading dengan Intersection Observer

üåê Delivery & CDN

  • Gunakan CDN untuk global delivery
  • Adaptive bitrate streaming
  • Progressive download
  • Video hosting (YouTube, Vimeo)

‚úÖ Yang Harus Dilakukan:
