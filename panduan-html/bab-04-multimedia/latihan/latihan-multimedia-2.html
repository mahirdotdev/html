🎬 Latihan Multimedia 2

Membuat Media Center Lengkap

Video • Audio • Iframe Integration

📋 Deskripsi Latihan

Dalam latihan ini, kalian akan membuat halaman Media Center yang menggabungkan video, audio, dan iframe dalam satu halaman terintegrasi.

🎯 Tujuan Pembelajaran:

  • Menggunakan tag <video> dengan multiple sources
  • Mengimplementasikan <audio> dengan kontrol
  • Menerapkan <iframe> untuk konten eksternal
  • Menerapkan responsive design untuk multimedia
  • Menggunakan atribut aksesibilitas

✅ Requirements Wajib:

  1. 1 Video Player dengan poster, controls, dan multiple sources
  2. 2 Audio Player dengan kontrol dan deskripsi
  3. 2 Iframe untuk konten eksternal (YouTube dan Maps placeholder)
  4. Responsive Design yang bekerja di mobile dan desktop
  5. Accessibility dengan title, alt, dan ARIA labels

🎥 Bagian 1: Video Player

Tugas: Buat video player dengan poster image dan multiple format sources

<video controls width="640" height="360" poster="URL_POSTER"> <source src="video.mp4" type="video/mp4"> <source src="video.webm" type="video/webm"> Browser tidak mendukung video HTML5. </video>

📝 KERJAKAN DI SINI

Tambahkan video player dengan:

  • Poster image (gunakan placeholder)
  • Controls attribute
  • 2 source formats (mp4 dan webm)
  • Fallback text
  • Ukuran yang responsive

🎵 Bagian 2: Audio Players

Tugas: Buat 2 audio player dengan tema berbeda dan deskripsi

<div class="audio-item"> <h4>Judul Audio</h4> <audio controls preload="metadata"> <source src="audio.mp3" type="audio/mpeg"> <source src="audio.ogg" type="audio/ogg"> Browser tidak mendukung audio. </audio> <p>Deskripsi audio...</p> </div>

📝 KERJAKAN DI SINI - AUDIO 1

Tema: Podcast/Radio Show

📝 KERJAKAN DI SINI - AUDIO 2

Tema: Musik/Lagu

🌐 Bagian 3: Iframe Integration

Tugas: Integrasikan 2 iframe untuk konten eksternal

<div class="iframe-container"> <h4>Judul Konten</h4> <iframe src="URL" width="560" height="315" title="Deskripsi" allowfullscreen loading="lazy"> </iframe> </div>

🗺️ Maps Integration

📝 KERJAKAN DI SINI

Tambahkan iframe "peta" dengan:

  • Placeholder untuk Google Maps
  • Title yang deskriptif
  • Ukuran yang sesuai

📱 Bagian 4: Responsive Design

Tugas: Tambahkan CSS untuk membuat semua media responsive

/* Tambahkan CSS ini di bagian <style> */ .responsive-media { width: 100%; max-width: 640px; height: auto; } .iframe-responsive { position: relative; width: 100%; height: 0; padding-bottom: 56.25%; /* 16:9 */ } .iframe-responsive iframe { position: absolute; top: 0; left: 0; width: 100%; height: 100%; } @media (max-width: 768px) { /* Mobile styles */ }

📝 CSS CHALLENGE

Tambahkan CSS responsive di bagian <style> untuk:

  • Video dan audio responsive
  • Iframe dengan aspect ratio 16:9
  • Media queries untuk mobile
  • Grid layout untuk audio players

💡 Tips & Panduan Pengerjaan

🎥 Untuk Video:

  • Gunakan https://sample-videos.com/ untuk sample video
  • Poster bisa dari https://via.placeholder.com/640x360
  • Jangan lupa atribut controls
  • Tambahkan preload="metadata" untuk optimasi

🎵 Untuk Audio:

  • Gunakan placeholder audio atau sample dari internet
  • Berikan judul dan deskripsi yang menarik
  • Pertimbangkan preload="metadata"
  • Tambahkan informasi durasi jika ada

🌐 Untuk Iframe:

  • YouTube: gunakan format embed URL
  • Selalu tambahkan title attribute
  • Gunakan loading="lazy" untuk performance
  • Buat responsive dengan CSS

📱 Untuk Responsive:

  • Test di berbagai ukuran layar
  • Gunakan max-width: 100% dan height: auto
  • Implementasikan aspect ratio untuk iframe
  • Pertimbangkan loading performance
