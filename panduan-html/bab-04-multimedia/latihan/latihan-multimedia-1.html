Membuat Galeri Foto Sederhana

📋 Deskripsi Latihan

Dalam latihan ini, kalian akan membuat galeri foto sederhana menggunakan tag <img> dengan berbagai atribut yang telah dipelajari.

🎯 Tujuan Pembelajaran:

  • Menggunakan tag <img> dengan benar
  • Menerapkan atribut src, alt, width, dan height
  • Menggunakan loading="lazy" untuk optimasi
  • Membuat layout grid sederhana dengan CSS
  • Menerapkan prinsip aksesibilitas

✅ Tugas yang Harus Diselesaikan:

  1. Buat 6 buah tag img dengan placeholder images dari via.placeholder.com
  2. Berikan alt text deskriptif untuk setiap gambar
  3. Gunakan ukuran yang konsisten (misalnya 300x200px)
  4. Tambahkan loading="lazy" untuk optimasi
  5. Buat layout grid 3 kolom dengan CSS
  6. Tambahkan deskripsi singkat di bawah setiap foto

💡 Tips Pengerjaan:

  • Gunakan warna yang berbeda untuk setiap placeholder agar terlihat menarik
  • Alt text harus mendeskripsikan isi gambar, bukan hanya "foto 1", "foto 2"
  • Test responsive dengan mengubah ukuran browser
  • Pastikan semua gambar memiliki ukuran yang konsisten
  • Gunakan tema yang berkaitan (misalnya: kegiatan sekolah, alam, teknologi)

⚠️ Hal yang Perlu Diperhatikan:

  • Jangan lupa menambahkan CSS untuk .gallery-container, .photo-item, dll.
  • Pastikan layout tidak berantakan di mobile
  • Alt text harus deskriptif dan bermakna
  • Test loading lazy dengan scroll halaman

🎉 Setelah Selesai

Simpan file sebagai galeri-foto-saya.html dan buka di browser untuk melihat hasilnya!
