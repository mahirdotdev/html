HTML5 Semantic Structure

Template ini menggunakan struktur HTML5 semantic yang benar:

<!DOCTYPE html>
<html lang="id">
<head>
    <!-- Meta tags, styles, scripts -->
</head>
<body>
    <header role="banner">...</header>
    <main role="main">
        <section>...</section>
        <article>...</article>
    </main>
    <aside role="complementary">...</aside>
    <footer role="contentinfo">...</footer>
</body>
</html>

Performance Optimization

Critical Resource Hints:

  • preconnect - Establish early connections
  • dns-prefetch - Resolve DNS early
  • preload - Priority loading for critical resources

Loading Strategy:

  • Critical CSS inlined in head
  • Non-critical CSS loaded asynchronously
  • JavaScript deferred until DOM ready
  • Images optimized with proper sizing
