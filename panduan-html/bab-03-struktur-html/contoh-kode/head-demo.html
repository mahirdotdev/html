🧠 Demo Tag <head> HTML

Halaman ini mendemonstrasikan penggunaan lengkap tag <head> dengan berbagai meta tags dan konfigurasi.

📋 Meta Tags yang Digunakan

✅ Basic Meta Tags (WAJIB)

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta http-equiv="X-UA-Compatible" content="IE=edge">

🔍 SEO Meta Tags

<title>Demo Tag Head HTML - Belajar Struktur HTML</title>
<meta name="description" content="Contoh lengkap penggunaan tag head...">
<meta name="keywords" content="HTML head, meta tags, SEO">
<meta name="author" content="Tim Pengajar HTML">

📱 Social Media Meta Tags

<meta property="og:title" content="Demo Tag Head HTML">
<meta property="og:description" content="Pelajari cara menggunakan...">
<meta name="twitter:card" content="summary">

🎯 Fungsi Setiap Bagian Head

1. Character Encoding

Tag <meta charset="UTF-8"> memungkinkan halaman ini menampilkan berbagai karakter:

  • 🌍 Emoji: 🚀 🎉 💻 📚 ⭐
  • © Simbol: © ® ™ € £ ¥
  • Aksen: café, naïve, résumé
  • 中文: 你好世界
  • العربية: مرحبا بالعالم

2. Viewport Meta

Tag viewport membuat halaman ini responsive di mobile device. Coba buka di smartphone!

3. Title Tag

Judul yang muncul di tab browser: "Demo Tag Head HTML - Belajar Struktur HTML"

4. Description Meta

Deskripsi yang akan muncul di Google search results dan social media preview.

5. Open Graph Tags

Ketika halaman ini dibagikan di Facebook, WhatsApp, atau LinkedIn, meta tags OG akan menentukan preview card.

🔬 Testing Head Tags

Browser DevTools Test:

  1. Tekan F12 untuk buka DevTools
  2. Pergi ke tab Elements
  3. Expand tag <head>
  4. Lihat semua meta tags yang tersedia

Network Test:

  1. Buka tab Network di DevTools
  2. Refresh halaman (F5)
  3. Lihat request untuk font Google Fonts
  4. Perhatikan preconnect yang mempercepat loading

Mobile Test:

  1. Klik icon Toggle Device Toolbar di DevTools
  2. Pilih device: iPhone, iPad, Samsung, dll
  3. Lihat bagaimana viewport meta bekerja
  4. Coba zoom in/out untuk test responsiveness

⚡ Performance Optimization

Head section ini menggunakan beberapa optimization techniques:

  • Preconnect: Mempercepat koneksi ke Google Fonts
  • DNS Prefetch: Resolve DNS lebih awal
  • Critical CSS: Inline styling untuk above-the-fold content
  • Font Display Swap: Mencegah FOIT (Flash of Invisible Text)

💡 Tips Performance:

  • Letakkan critical CSS di head
  • Defer non-critical JavaScript
  • Gunakan preconnect untuk external resources
  • Optimize meta tags untuk SEO
  • Minimize JavaScript di head section
