💡 Tips Penggunaan

  • <br>: Gunakan untuk line break yang memang diperlukan (alamat, puisi, kontak)
  • <hr>: Gunakan untuk memisahkan section atau topik yang berbeda
  • Jangan gunakan multiple <br> untuk spacing - gunakan CSS
  • Jangan gunakan <hr> hanya untuk dekorasi visual
