2. Atribut Form Penting

Tag <form> memiliki beberapa atribut penting yang mengatur perilaku form:

Atribut Deskripsi Contoh Nilai Fungsi
action URL tujuan pengiriman data /submit, https://api.example.com Menentukan kemana data dikirim
method Metode HTTP untuk pengiriman get, post GET untuk data publik, POST untuk data sensitif
enctype Tipe encoding data application/x-www-form-urlencoded, multipart/form-data multipart/form-data untuk upload file
target Target pembukaan response _self, _blank Mengatur dimana response ditampilkan

3. Perbedaan Method GET dan POST

üì§ Method GET

Karakteristik:

  • Data tampil di URL
  • Batas data ~2000 karakter
  • Tidak aman untuk data sensitif
  • Bisa di-bookmark

üîê Method POST

Karakteristik:

  • Data tersembunyi dari URL
  • Tidak ada batas ukuran data
  • Lebih aman untuk data sensitif
  • Tidak bisa di-bookmark
